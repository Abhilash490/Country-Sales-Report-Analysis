# Global Sales Analysis Dashboard - Power BI

## Overview
This project is a Power BI Sales Dashboard created to analyze country-wise sales performance using interactive visualizations and maps. The dashboard helps in understanding revenue, profit, and sales trends across different regions.

## Features
- Country-wise sales analysis using maps
- Revenue and profit KPI cards
- Sales trend visualization
- Interactive filters and slicers
- Regional performance insights

## Tools Used
- Power BI
- Excel / CSV Dataset
- DAX
- Data Visualization Techniques

## Dashboard Visuals
- Map Charts
- Bar Charts
- Line Charts
- Pie Charts
- KPI Cards

## Workflow
1. Data Collection
2. Data Cleaning
3. Data Transformation
4. Dashboard Creation
5. Sales Analysis

## Insights
- Identified top-performing countries
- Compared regional sales performance
- Analyzed sales growth trends
